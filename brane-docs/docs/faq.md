# Frequently Asked Questions (FAQ)

### What is Brane?
Brane is a programmable, site-agnostic framework designed to run distributed workflows securely over multiple computing sites.

### Where should I start?
* If you are a domain scientist wanting to run workflows or register datasets, start with the [Quick Start for Scientists](brane-user-guide/02-quick-start-scientists.md).
* If you are looking to integrate your own custom code as a package, check out the [Software Engineers Guide](brane-user-guide/04-software-engineers.md).

### How do I deploy a local Brane instance?
You can find the structural deployment requirements and steps in the [System Engineers: Deploying Brane Guide](brane-user-guide/03-system-engineers-deploying.md).
