# 1. Tutorial Overview

This chapter lists the tutorial materials covered in this book.

We group sessions by purpose:

- **Tutorials and demos**  
  Hands‑on exercises and presentations introducing Brane to new users, typically at conferences, hospitals, or research meetings.

Currently included:

- **ICT.OPEN 2023** (20‑04‑2023)  
  First public tutorial introducing the Brane framework:
  - Role‑playing as software engineer and scientist.
  - Building a simple package (Hello, World).
  - Writing and running a workflow (Disaster Tweets pipeline).

- **UMC Utrecht demo and workshop** (23‑10‑2023)  
  Tutorial as part of a proof‑of‑concept on pseudonymised patient data:
  - Introductory talk at a SIG meeting.
  - Hands‑on session: packages, workflows, remote execution.
  - Technical presentation of the Brane PoC setup.

Use the links above or the sidebar to navigate to the specific tutorial.
