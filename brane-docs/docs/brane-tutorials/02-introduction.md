# 2. Introduction

>The first tutorial introducing users to the Brane framework was given at **ICT.OPEN 2023**, a conference aiming to bring academia and industry together.

### 2.1. Purpose

Participants experience:

- The **software engineer** role:
  - Writing and packaging code as Brane packages (ECUs).
  - Building and testing packages locally.

- The **scientist** role:
  - Writing workflows that call packages.
  - Running workflows locally and (optionally) remotely.

The tutorial is written for Brane framework **version 2.0.0**.

### 2.2. Schedule

The tutorial is structured as:
